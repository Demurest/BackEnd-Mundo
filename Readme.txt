Instrucciones para el BackEnd

Primero hay que instalar laragon, luego
en la carpeta root de laragon hacer git clone del repositorio.

Segundo, presionar 'StartAll' de la interfaz de laragon, para activar la BDD.

Tercero, abriendo un cmd dentro de 'Proyecto' utilizar los siguiente comandos:

php artisan migrate:fresh

php artisan db:seed

con esto ya tenemos creada la BDD y poblada.
